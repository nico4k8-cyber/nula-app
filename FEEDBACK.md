## 🔧 Технические проблемы

### 1. Музыка на DragonSplashScreen
**Проблема**: Музыка не воспроизводится на первом экране
**Причины**:
- Добавлены `autoPlay` и `muted` атрибуты
- Обработчик для размучивания при взаимодействии пользователя добавлен
- Но эффект не виден в браузере

**Решение**: Нужно проверить:
1. Загружается ли файл `/audio/dragon-flying.mp3` (статус 200?)
2. Есть ли ошибки в консоли браузера
3. Может быть требуется мьютирование не на элементе, а в коде?

### 2. DragonSplashScreen не показывается
**Проблема**: При загрузке сразу переходим на OnboardingScreen
**Причина**: Флаг `hasSeenDragonSplash` остаётся true в localStorage даже после очистки
**Следствие**: Пользователь не может повторно увидеть красивую заставку

### 3. XP система непонятна
**Проблема**: Показываем числа (20, 50 XP), но детям не ясно что это значит
**Решение**: Переделать на задачи:
- "Доступно! ✓" → "Доступно уже сейчас"
- "Откроется при 20 XP" → "Нужно решить 5 простых задач"
- "нужно ещё 20" → "нужно решить ещё 5 простых"
