# 🎨 ПАРАДОКС V2 — ЭКРАНЫ И ФЛОУ

**Версия:** 2.0 (Chat-Based TRIZ Learning)
**Дата:** 28 Mar 2026
**Целевой размер:** 375×812px (iPhone 12 размер)
**Доступность:** WCAG AA (контраст, touch-safe 44px)

---

## 📱 ПОЛНЫЙ ФЛОУ СЕССИИ

```
HOME SCREEN
    ↓
SELECT PROBLEM (или автоматический)
    ↓
PROBLEM IMAGE DISPLAY (30 сек)
    ↓
CHAT SCREEN — Stage 1-6 GINA (4-5 минут)
    ├─ Stage 1: Ideal Final Result
    ├─ Stage 2: Identify Contradiction
    ├─ Stage 3: Separate in Space/Time
    ├─ Stage 4: Reverse the Problem
    ├─ Stage 5: Use Resources
    └─ Stage 6: Solution & Insight
    ↓
SOLUTION CONFIRMATION SCREEN
    ↓
COMPLETION SCREEN (XP, Unlock Next)
    ↓
PARENT REPORT (async via email)
```

## 🎨 ДИЗАЙН СИСТЕМА

**Цвета:**
- Primary: #10B981 (green, успех)
- Secondary: #6B7280 (gray, inactive)
- Accent: #3B82F6 (blue, Уголёк)
- Danger: #EF4444 (red, ошибка)
- Background: #FFFFFF (white)
- Text: #111827 (very dark gray)

**Типография:**
- Заголовки: 28px, bold, sans-serif
- Основной текст: 16px, regular
- Подтекст: 14px, light gray
- Chat: 16px, left-aligned

**Иконки:**
- Emoji для задач (🪞 🎒 💡 🎮)
- Material Icons для UI (back, home, settings)
- Custom Уголёк спрайт

**Animations:**
- Page transition: fade (300ms)
- Button press: scale (100ms)
- Chat message: slide in from bottom (200ms)
- Confetti: на completion screen (1s)
- Streak badge: pulse когда растёт

**Accessibility:**
- Все кнопки 44×44px minimum (touch-safe)
- Контраст AAA (4.5:1 для text)
- Alt text для всех изображений
- Voice input поддержка на всех stages
- Keyboard navigation (Tab, Enter, Escape)
